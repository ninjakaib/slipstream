- private let baseURL = "https://vpic.nhtsa.dot.gov/api/vehicles"
- private var makesCache: [VehicleMake]? = nil (cache makes since list is static)

Define response models:
- NHTSAMakesResponse: Codable with Count: Int, Results: [VehicleMake]
- VehicleMake: Codable, Identifiable with MakeId: Int, MakeName: String, var id: Int { MakeId }
- NHTSAModelsResponse: Codable with Count: Int, Results: [VehicleModel]
- VehicleModel: Codable, Identifiable with Make_ID: Int, Make_Name: String, Model_ID: Int, Model_Name: String, var id: Int { Model_ID }

Methods:
- func getYears() -> [Int] — returns Array((1980...(currentYear + 1)).reversed()) where currentYear from Calendar.current
- func getMakes() async throws -> [VehicleMake] — fetches from /GetMakesForVehicleType/car?format=json, caches result, sorts by MakeName. Append custom entry VehicleMake(MakeId: -1, MakeName: "Other / Custom") at end per D-05.
- func getModels(make: String, year: Int) async throws -> [VehicleModel] — fetches from /GetModelsForMakeYear/make/{encodedMake}/modelyear/{year}?format=json, sorts by Model_Name. Append custom entry VehicleModel(Make_ID: -1, Make_Name: make, Model_ID: -1, Model_Name: "Other / Custom") at end per D-05.

Use URLSession.shared.data(from:) with async/await. Handle URL encoding for make names with spaces.

Per D-04: Use NHTSA vPIC API for year/make/model data.
Per D-05: Include "Other / Custom" option at end of every picker.
